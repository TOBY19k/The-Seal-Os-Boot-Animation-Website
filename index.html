Seal Os Boot Animation Installer

A zip flash file that installs Seal Os boot animation on the s7. Fully based on Dynamic Installer. For instructions to install go here.

How To Install

  1. Download Seal Os Boot Animation Installer
  2. Go into custom recovery (Twrp , Orange Fox , SHRP , Lineage Recovery, or pixelexperience Recovery )
    • Note the Orange Fox link will only work if you are in the S7 AOSP-like ROMs (Early-Test and Discussion) telegram group .
  3. Find the downloaded zip and flash it
  4. Reboot into system
